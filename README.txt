# The Logic analyser

This tool can be run on http://pravar.pythonanywhere.com/ or can be run locally by following the steps below:

To run locally it is recommended you have Python 3.8 and pip installed.

The required libraries are in the 'requirements.txt'. To install these, try: 'pip install -r requirements.txt' or 'pip3 install -r requirements.txt'

Then run python3 flask_app.py

On a browser go to http://127.0.0.1:5000/ to access the tool
